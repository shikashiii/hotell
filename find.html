Instruktioner som kan hjälpa dig att hitta hotellet

  1. Gå in från norr
  2. Leta efter den stora stenen med skatorna på
  3. Ge skatorna ett stycke bröd på cirka 150g, kan vara vilket bröd som helst
  4. Följ efter skatan med blixtrande lila ögon till den stannar vid en viss kantarell
  5. Knacka dig själv på huvudet 3 gånger och säg "Jag söker tak över huvudet, och mat på min tallrik"
